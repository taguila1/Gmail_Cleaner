# Extension Icons

This directory should contain the following icon files:
- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels  
- `icon128.png` - 128x128 pixels

## Creating Icons

You can create simple icons using:
1. Online icon generators
2. Image editing software (Photoshop, GIMP, etc.)
3. SVG to PNG converters

For a quick start, you can use a simple colored square or create an icon with:
- A mail/envelope symbol
- A trash/cleanup symbol
- A combination of both

The icons should be PNG format with transparency support.

## Temporary Solution

Until you create proper icons, you can:
1. Use placeholder images
2. The extension will still work but may show a default Chrome extension icon
3. Create simple colored squares as placeholders
